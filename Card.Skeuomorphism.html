CRX Finder

This tool is used to find CRX files on the Chrome Web Store.

How to use:

1. Enter the ID of the Chrome Web Store item you want to find.
2. Click the "Find" button.
3. Wait for the tool to finish.
4. Click the "Download" button to download the CRX file.
